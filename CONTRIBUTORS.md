## How to Contribute

We welcome contributions! Here's how you can help:

1. **Content** - Improve or expand course modules
2. **Translations** - Help translate content to Urdu or other languages
3. **Bug Fixes** - Fix issues and improve stability
4. **Features** - Suggest and implement new features
5. **Documentation** - Improve guides and documentation

See [docs/development/contributing.md](docs/development/contributing.md) for detailed contribution guidelines.
