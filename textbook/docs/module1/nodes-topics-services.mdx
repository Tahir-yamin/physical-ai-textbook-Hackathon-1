      <h1>Nodes, Topics, and Services</h1>
      
      <h2>ğŸ¯ Understanding ROS 2 Communication</h2>
      
      <p>ROS 2 provides multiple communication patterns to suit different robotics scenarios. The three primary mechanisms are:</p>
      
      <ol>
        <li><strong>Topics</strong>: Asynchronous, publish-subscribe (many-to-many)</li>

      <h2>ğŸ“¡ Topics: Publish-Subscribe Pattern</h2>
      
      <h3>What Are Topics?</h3>
      
      <p><strong>Topics</strong> are named buses over which nodes exchange messages. They implement a <strong>publish-subscribe</strong> pattern:</p>
      
      <ul>
        <li>Publishers send messages to a topic</li>
        <li>Subscribers receive messages from a topic</li>
        <li>Decoupled: Publishers don't know who's subscribing</li>
        <li>Asynchronous: Non-blocking communication</li>
      </ul>
      
      <h3>Use Cases</h3>
      
      <p>Topics are ideal for:</p>
      <ul>
        <li>âœ… Streaming sensor data (camera images, laser scans)</li>
        <li>âœ… Broadcasting robot state</li>
        <li>âœ… High-frequency data streams</li>
        <li>âœ… One-to-many or many-to-many communication</li>

import rclpy
from rclpy.node import Node
from example_interfaces.srv import AddTwoInts
import sys

class AddTwoIntsClient(Node):
    def __init__(self):
        super().__init__('add_two_ints_client')
        self.client = self.create_client(AddTwoInts, 'add_two_ints')

        # Wait for service to be available
        while not self.client.wait_for_service(timeout_sec=1.0):
            self.get_logger().info('Service not available, waiting...')

    def send_request(self, a, b):
        request = AddTwoInts.Request()
        request.a = a
        request.b = b

        future = self.client.call_async(request)
        return future

    node = AddTwoIntsClient()

    future = node.send_request(int(sys.argv[1]), int(sys.argv[2]))
    rclpy.spin_until_future_complete(node, future)

    response = future.result()
    node.get_logger().info(f'Result: {sys.argv[1]} + {sys.argv[2]} = {response.sum}')

    node.destroy_node()
