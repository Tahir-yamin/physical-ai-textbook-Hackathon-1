      <h2>ğŸ›ï¸ Core Concepts</h2>
      
      <h3>1. Nodes</h3>
      
      <p><strong>Nodes</strong> are the fundamental computing units in ROS 2. Each node should perform a single, well-defined task:</p>
      
      <ul>
        <li><code>camera_node</code>: Captures images from a camera</li>
        <li><code>object_detector</code>: Processes images to detect objects</li>
        <li><code>motor_controller</code>: Controls robot motors</li>
        <li><code>path_planner</code>: Plans navigation paths</li>
      </ul>
      
      <h3>2. Computational Graph</h3>
      
      <p>Nodes communicate to form a <strong>computational graph</strong>:</p>
      
      <pre>{`
[Camera Node] --image--> [Object Detector] --objects--> [Decision Maker]
                                                              |
                                                              v
                                                     [Motor Controller]
      `}</pre>
      
      <h3>3. Discovery Mechanism</h3>
      
      <p>ROS 2 uses <strong>DDS Discovery</strong> for automatic peer finding:</p>
      
      <ul>
        <li>No central master (unlike ROS 1)</li>
        <li>Nodes automatically discover each other</li>
        <li>Resilient  to network changes</li>
        <li>Supports multi-robot systems</li>
      </ul>
